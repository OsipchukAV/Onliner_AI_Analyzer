# Название проекта
Onliner AI Analyzer (Intellectual Scale Core) — Масштабируемая интеллектуальная система сбора и анализа данных торговых площадок.
Описание программы:
Программа представляет собой универсальное программное ядро для автоматизации мониторинга вторичных рынков. Система выполняет полный цикл обработки объявлений:
Парсинг: Агрессивный сбор текстовой и визуальной информации с торговых площадок (на примере Onliner Baraholka).
ИИ-анализ: Использование больших языковых моделей (LLM Gemini) для извлечения атрибутов товара (бренд, модель, состояние) из неструктурированного текста и формирования унифицированного JSON-паспорта.
Векторная память: Формирование мультимодальных эмбеддингов (текст + фото) с помощью модели CLIP для реализации семантического поиска «по смыслу».
Интерфейс: Веб-приложение для оперативного анализа ссылок и управления глобальной базой знаний.
Проект спроектирован по модульному принципу, что позволяет расширять систему в ширину, подключая новые торговые площадки без изменения логики анализа и хранения.
# Требования
Язык программирования: Python 3.10+
Менеджер пакетов: pip
Основные библиотеки:
streamlit (интерфейс)
chromadb (векторная база данных)
sentence-transformers (модели CLIP)
beautifulsoup4 (парсинг)
openai (взаимодействие с OpenRouter API)
# Инструкция по запуску
Клонировать репозиторий.
Установить зависимости:
code
Bash
pip install -r requirements.txt
Создать файл .env в корневом каталоге и добавить ваш API ключ:
code
Env
OPENROUTER_API_KEY=ваш_ключ_здесь
Запустить веб-приложение:
code
Bash
streamlit run web_app.py
